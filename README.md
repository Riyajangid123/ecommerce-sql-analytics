# ecommerce-sql-analytics
This project analyzes e-commerce sales data using SQL to uncover insights related to revenue, products, and customer behavior. The goal is to support data-driven business decisions through structured queries and analysis.
# Skills & Tools
SQL, MySQL / PostgreSQL, Data Analysis, Joins, Subqueries, Aggregations, Group By, Order By, CTEs
# Key Objectives
Analyze overall and monthly sales performance
Identify top-selling products and categories
Understand customer purchase behavior
Measure average order value and repeat customers
Evaluate payment method trends
# Key Analysis Performed
Revenue and sales trend analysis
Product and category performance analysis
Customer segmentation (repeat vs one-time buyers)
Average Order Value (AOV) calculation
Payment method usage analysis
# Conclusion
This project demonstrates hands-on SQL expertise and the ability to transform raw e-commerce data into meaningful business insights, aligning with Data Analyst and Business Analyst role requirements.
